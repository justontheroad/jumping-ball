# Three.js 无限轨道弹跳球 - 项目迭代计划

## 项目概述

基于 Three.js 的 3D 无限轨道弹跳球游戏，采用星际穿越主题。玩家控制一个弹跳球在无限生成的轨道上前进，需要匹配颜色才能安全着陆，最终目标是进入黑洞完成关卡。

---

## 核心架构

### 1. 物理系统

**当前实现：**
- 基于真实时间的物理计算（`performance.now()` + `dt`）
- Y 轴重力：`GRAVITY_Y = -14.7 m/s²`（1.5倍真实重力）
- 弹跳高度固定：`INIT_HEIGHT = 3.0 m`
- 弹跳周期公式：`T = 2 * sqrt(2h/g)`
- Z 方向速度动态计算：`velZ = |dist| / BOUNCE_PERIOD`
- 轨道间距随机范围：`[GAP_MIN=3.75, GAP_MAX=7.5]`

**迭代方向：**
- [ ] 考虑添加不同重力环境（如低重力星球）
- [ ] 优化物理参数可配置化
- [ ] 添加球体旋转物理效果

### 2. 轨道系统

**当前实现：**
- 类型1：长直轨道（单块长板，一种颜色）
- 类型2：两块方形轨道（二色各一）
- 类型3：三块方形轨道（三色各一）
- 类型4：加速轨道（白色发光，触发3倍重力加速）
- 轨道淡出销毁机制（FADE_START=3, FADE_END=12）
- 可见深度：`VISIBLE_DEPTH = 120`

**迭代方向：**
- [ ] 添加新轨道类型（如移动轨道、旋转轨道）
- [ ] 轨道生成算法优化（难度曲线）
- [ ] 轨道视觉特效增强（发光、纹理动画）

### 3. 颜色系统

**当前实现：**
- 三种颜色：粉色(pink)、黄色(yellow)、蓝色(blue)
- 球颜色与轨道颜色匹配机制
- 长直轨道改变球颜色
- 方形轨道需要颜色匹配才能安全着陆
- 异色碰撞触发碎裂死亡

**迭代方向：**
- [ ] 添加更多颜色选项
- [ ] 颜色变化动画过渡
- [ ] 特殊颜色道具（如彩虹色）

### 4. 加速系统

**当前实现：**
- 加速轨道（类型4）：白色发光长直轨道
- 加速持续：10秒
- 重力倍数：3倍
- 蓄力机制：首次踩到原地弹跳蓄力
- 加速背景粒子系统：8000粒子，螺旋隧道分布
- 粒子颜色跟随球颜色变化

**迭代方向：**
- [ ] 加速效果视觉增强

### 5. 黑洞系统

**当前实现：**

- 事件视界：`BH_EVENT_HORIZON = 1.2`
- 吸积盘着色器效果（旋涡纹理）
- 被吸入粒子系统（300粒子）
- 渐入缩放动画（2秒）

## 开发规范

### 代码风格
- 常量使用大写下划线命名（如 `GRAVITY_Y`）
- 函数使用驼峰命名（如 `onLandUpdateVelZ`）
- 变量使用驼峰命名（如 `ballX`, `velZ`）

### 物理参数
- 所有距离单位：米
- 时间单位：秒
- 角度单位：弧度

### 性能考量
- 共享几何体（`longGeo`, `sqGeo`）
- 材质实例管理（独立淡出）
- 粒子数量控制（背景4000，加速8000）
- 着色器优化（uniform 控制）
