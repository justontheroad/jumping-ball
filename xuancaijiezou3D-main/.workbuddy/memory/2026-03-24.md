# 2026-03-24 工作日志

## Three.js 无限轨道弹跳球（three-sphere.html）

### 重构：基于真实时间的物理系统
- 废弃了基于帧的物理计算（`GRAVITY=-0.003`, `TRACK_SPEED=-0.08`），改用真实世界时间 `performance.now()` + `dt`
- Y 轴重力改为真实值 `GRAVITY_Y = -9.8 m/s²`，弹跳周期通过公式计算：`T = 2 * sqrt(2h/g)`
- 弹跳高度固定 `INIT_HEIGHT = 3.0`，通过最高点修正防止浮点累积
- Z 方向速度每次落地后根据下一块板的距离重新计算：`velZ = |dist| / BOUNCE_PERIOD`，保证球每次正好落在板中心
- 相邻轨道间距在 `[GAP_MIN=2.0, GAP_MAX=4.5]` 范围内随机

### 轨道管理简化
- 从"段管理"改为"单块管理"：每块轨道板独立创建/移动/淡出/销毁
- 每块板有独立的材质实例，支持独立淡出
- 销毁时材质也 `dispose()` 防止泄漏

### 修复 TDZ 错误
- `TRACK_SPEED` 在声明前被引用导致整个脚本静默崩溃（const 暂时性死区）
- 将常量声明移到使用位置之前
